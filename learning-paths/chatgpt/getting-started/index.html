Getting Started


ChatGPT is already a powerful tool out of the box. But with just a few small adjustments, you can dramatically improve how it responds — making it more accurate, more helpful, and better suited to your workflow.

This guide shows you how to customize ChatGPT settings to match your communication style and use Customs GPTs / Projects to organize your work.


Customize ChatGPT

Start by telling ChatGPT how you want it to behave. This takes just a minute and changes everything.

1. Open Customization

In the ChatGPT web app:

  • Click your name or profile icon (top right corner)
  • Select “Customize ChatGPT”

This feature is available to all users, even on the free plan.

2. Set Behavior Preferences

You'll see two key customization fields:

What would you like ChatGPT to know about you?

Use this to define:

  • Your name, profession, and goals
  • Preferred tone or language (e.g. casual, formal, technical)
  • Whether you're working solo or on a team

Example:

I'm a product designer working at a startup. I prefer clear, concise explanations and visual thinking.

How would you like ChatGPT to respond?

Set formatting, tone, and rules of engagement.

Example:

Use bullet points when possible. Avoid buzzwords. Keep answers under 300 words unless I ask for more detail.

3. Enable or Disable Memory (Pro Feature)

If you’re on ChatGPT Plus:

  • Go to Settings > Personalization
  • Toggle Memory on or off
  • You can also delete stored memory at any time

Tip: If you're working across multiple projects, consider keeping Memory off and using Custom Instructions or Projects for control.
